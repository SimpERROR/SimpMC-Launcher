快速开始

1

下载最新版本

前往 GitHub Releases 页面，下载对应您操作系统的安装包。

2

安装启动器

Windows 用户下载 .exe 或 .zip 版本，macOS 用户下载 .dmg 版本。

3

登录微软账号

首次使用需要登录您的微软账号。我们仅使用 OAuth 方式，不会获取您的密码。

4

开始游戏

选择想要的游戏版本，点击启动按钮即可开始您的 Minecraft 冒险！

隐私政策

🔒 您的隐私安全是我们的首要任务

  • 身份验证：SimpMC 仅通过微软官方 OAuth 2.0 接口完成身份验证，我们无法也不会获取您的微软账号密码。
  • 本地存储：您的登录令牌（Token）仅保存在本地设备中，用于后续启动游戏时的身份验证。
  • 数据收集：我们不会收集、存储、上传您的任何个人身份信息至任何远程服务器。
  • 游戏数据：游戏版本文件、模组配置等数据均存储在您本地选择的游戏目录中。
  • 开源透明：作为开源项目，我们的全部代码均在 GitHub 上公开，接受社区审查。
  • 第三方服务：我们仅使用 Mojang/微软官方服务器进行游戏验证，不使用任何第三方登录服务。
